Opensea plugin for [frog üê∏](https://ohmyfrog.dev). Allows you to create tools for getting NFT collection data from Opensea.

## Installation
```
npm install @frog-sdk/plugin-opensea
```

## Usage

```typescript
import { opensea } from "@frog-sdk/plugin-opensea";

const plugin = opensea({
    apiKey: process.env.OPENSEA_API_KEY as string,
});
```
